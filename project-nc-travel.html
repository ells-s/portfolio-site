Overview

NC Travel is a mobile app built using React Native. It was designed to simplify the travel planning experience. It has various search features. You can search for: flights, hotels and activities. Filters can also be applied to refine the search. Travel options can then be selected and added to a itinerary so all the plans are in one place.

This project was built as a team. We not only use skills learnt throughout the bootcamp, but also learnt new technologies such as React Native.

Technologies Used

  • React Native
  • TypeScript
  • Firebase
  • Firestore
  • Amadeus API
  • Jest
