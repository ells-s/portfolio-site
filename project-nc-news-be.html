Overview

The NC News API is a RESTful backend built using Node.js, Express, and PostgreSQL. It powers my frontend NC News application by serving structured data related to news content. The API enables CRUD operations on topics, articles, users, and comments, simulating the functionality of a news platform.

Technologies Used

  • Node.js
  • Express
  • PostgreSQL
  • dotenv
  • Jest
