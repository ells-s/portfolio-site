Overview

NC News is a fully responsive frontend application that emulates a modern news website. Built using React, it allows users to browse, sort, and interact with news articles fetched from a custom-built backend API. This project showcases my skills in React component architecture, API integration, and frontend design with a focus on usability and clean UI.

This project is the frontend companion to my Node.js PostgreSQL, Express backend project which handles all data operations via a RESTful API.

Technologies Used

  • React
  • React Router
  • Netlify
  • Custom backend API (Node.js, Express, PostgreSQL)
  • CSS
